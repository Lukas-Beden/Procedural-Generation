# Génération Procédurale & Portails

Jeu d'exploration en Unity où les niveaux sont générés procéduralement. Le joueur récolte des pots de peinture pour dessiner un portail sur le sol et passer au niveau suivant.

Les niveaux se succèdent dans un ordre fixe : plaines → désert → neige.

## Mécaniques

- Génération procédurale des niveaux
- Collecte de pots de peinture
- Portails dessinés par le joueur

## Stack

- Unity 6000.1.17f1 (C#)

## Lancer le projet

```bash
git clone https://github.com/Lukas-Beden/Procedural-Generation.git
```

Ouvrir dans Unity Hub et lancer la scène principale.
